# 🎮 中国游戏在线版 - 联机对战

在线中国象棋和斗地主，支持多人联机！

## 🌐 在线试玩

- 🀄 中国象棋: https://your-service.onrender.com/
- 🃏 斗地主: https://your-service.onrender.com/

## ✨ 功能特点

### 斗地主
- ✅ 3人实时联机对战
- ✅ 房间系统
- ✅ 叫分、抢地主
- ✅ 完整出牌逻辑
- ✅ 游戏结算

### 中国象棋
- ✅ 2人实时联机对战
- ✅ 房间系统
- ✅ 完整象棋规则

## 🛠 技术栈

- Node.js + Express
- Socket.io 实时通信
- HTML5 Canvas

## 📦 本地运行

```bash
npm install
npm start
```

访问 http://localhost:10000

## 🚀 部署到 Render

1. Fork 本仓库
2. 在 Render 创建 Web Service
3. 连接 GitHub 仓库
4. 自动部署
